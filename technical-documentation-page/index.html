Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the foundational language used to create the structure of web pages. It uses a system of tags to define elements like text, links, images, and more. Each element provides a specific function, allowing browsers to interpret and render the content correctly. HTML works in combination with CSS and JavaScript to create fully functional web pages.

Important HTML tags

Some essential tags that define the structure and content of web pages:

  • <div>: Defines a section or a container in a document.
  • <header>: Represents the header section of a page.
  • <footer>: Defines the footer section with contact info, links, etc.
  • <section>: Groups related content within a page.
CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is a language that controls the presentation of HTML elements. It allows developers to change colors, layouts, fonts, and positioning to create visually appealing pages. CSS makes it easier to keep styles consistent across multiple pages of a website by defining a set of rules in one place.

Selectors and properties

Selectors target HTML elements, and properties define how these elements are styled:

  • Element selector: Targets HTML elements by name, e.g., h1.
  • Class selector: Targets elements with a specific class, e.g., .example.
  • ID selector: Targets a unique element by its ID, e.g., #header.

The Box Model

The CSS Box Model describes how elements are spaced and sized:

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: A line around the element.
  • Margin: Space outside the border, separating the element from others.
Introduction to JavaScript

What is JavaScript?

JavaScript is a programming language that allows you to add interactivity to web pages. It enables tasks like form validation, creating dynamic content, or responding to user inputs. JavaScript runs on the client side (in the browser) and works closely with HTML and CSS to enhance user experience.

Variables and data types

Variables store data that can be used and modified in your script:

  • let: Defines a variable that can be changed, e.g., let score = 0;.
  • const: Defines a constant value, e.g., const pi = 3.14;.
  • Data types: Common types include numbers, strings, booleans, and arrays.

Functions and loops

Functions help organize code into reusable blocks, while loops allow repetitive actions:

  • Function: A block of code that runs when called, e.g., function add(a, b) { return a + b; }.
  • Loop: Repeats actions, e.g., for (let i = 0; i < 10; i++) { ... }.

DOM manipulation

The Document Object Model (DOM) represents the structure of an HTML page, which JavaScript can modify:

  • getElementById: Selects an element by its ID, e.g., document.getElementById('header').
  • querySelector: Selects elements using a CSS selector, e.g., document.querySelector('.class').
Responsive Design

What is responsive design?

Responsive design ensures that web pages adapt to different screen sizes, from desktops to mobile devices. A responsive site automatically resizes and repositions elements to improve readability and usability on various devices. This is achieved using flexible grids, layouts, and media queries.

  • @media (max-width: 600px): Styles for screens narrower than 600 pixels.
  • @media (max-width: 1024px): Styles for screens wider than 1024 pixels.

Mobile-first approach

In this approach, developers design for smaller screens first, then scale up for larger devices. This ensures that the website works well on mobile devices and offers a more streamlined experience.

Flexbox and Grid

These CSS techniques help create flexible layouts:

  • Flexbox: Aligns and distributes space within a container, even when screen size changes.
  • Grid: Divides the page into rows and columns, creating complex layouts with ease.
Web Development Tools

What are web development tools?

Web development tools help developers build, test, and debug websites efficiently. These tools range from browser developer tools to version control systems like Git, making the coding process smoother and more manageable.

Browser developer tools

Browsers like Chrome and Firefox offer built-in tools for debugging and inspecting code:

  • Elements: View and edit HTML and CSS in real-time.
  • Console: Run JavaScript and view error messages.

Code editors

Developers use specialized editors to write and manage code:

  • VS Code: A popular editor with syntax highlighting, extensions, and Git integration.
  • Sublime Text: Lightweight, fast, and supports multiple languages.

Git and GitHub

Version control is essential for tracking changes and collaborating on projects:

  • Git: A distributed version control system for tracking changes in code.
  • GitHub: A platform for hosting repositories and collaborating with others.
